# Ruby on Rails API with Vue.js

#### Used in this build

- `Rails 5.2.2`
- `Ruby 2.6`
- Gem `bcrypt 3.1.7`
- Gem `rack-cors`
- Gem [`redis 4.1.0`](https://github.com/redis/redis-rb)
- Gem [`jwt-sessions`](https://github.com/tuwukee/jwt_sessions)

### Running the app
- `rails db:create`
- `rails db:migrate`
- `rails s`
